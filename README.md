# Objective

The main objective of this project is to build a **simple credit risk evaluation system** using Python.

The system evaluates important factors such as:

- Customer income
- Credit history
- Existing loans
- Payment behavior

The system helps to:

- Analyze basic customer **financial information**
- Calculate a **Risk Score**
- Estimate the **Probability of Default**
- Provide a **Loan Approval Recommendation**

This project helps students understand how **banks analyze loan applications** using simple data analysis techniques.


# Problem Statement

Banks receive thousands of loan applications from customers.
Before approving a loan, banks must check whether the customer will be able to **repay the loan on time**.
Manually analyzing each customer can be time-consuming.

Therefore, we build a **Python-based Credit Risk Model** that can:

1. Analyze customer financial data
2. Evaluate risk factors
3. Predict whether a loan should be approved or rejected


# Input Parameters

The system collects the following information from the customer:

**1. Customer Income**

Monthly or annual income of the customer.

**2. Credit History**

Customer's past loan repayment behavior (Good / Average / Poor).

**3. Existing Loans**

Number of active loans the customer currently has.

**4. Payment Behavior**

Whether the customer usually pays bills on time (OnTime / Late / VeryLate).

# Output Results

After analyzing the data, the system generates the following outputs:

**1. Risk Score**

A numerical value representing the overall risk level.

**2. Loan Decision**

      - Approved
      - Review Manually
      - Rejected

**3. Probability of Default**

Indicates the likelihood that the customer may fail to repay the loan.


# Conclusion

The **Credit Risk Intelligence Model** is a beginner-friendly project that demonstrates how **Python programming can be used in the banking and finance sector**.
This project helps students understand the **basic logic behind loan approval systems** used in financial institutions.
